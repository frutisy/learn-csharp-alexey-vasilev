10. Напишите программу с классом, у которого есть поле, являющееся ссылкой 
на целочисленный массив. При создании объекта массив заполняется случайными числами. 
Переопределите в классе метод ToString() так, чтобы метод возвращал текстовую строку 
со значениями элементов массива. Также строка должна содержать информацию о количестве 
элементов массива и среднем значении для элементов массива (сумма значений элементов, 
деленная на количество элементов).
