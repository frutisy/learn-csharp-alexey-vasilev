10. Напишите программу со статическим методом для вычисления косинуса.
Используйте формулу cos(x) = 1 - x^2/2! + x^4/4! - x^6/6! + ... + (-1)^n * x^2n / (2n)!.
В классе также должны быть статические методы для вычисления гиперболического синуса
sh(x) = x + x^3/3! + x^5/5! + ... + x^2n+1 / (2n+1)! и гиперболического косинуса
ch(x) = 1 + x^2/2! + x^4/4! + ... + x^2n / (2n)!.
