2. Напишите программу со статическим методом, которым вычисляется
сумма квадратов натуральных чисел 1^2 + 2^2 + 3^2 + ... + n^2. 
Число n передается аргументом методу. Предложите версию метода с рекурсией 
и без рекурсии. Для проверки результата можно использовать формулу 
1^2 + 2^2 + 3^2 + ... + n^2 = n(n+1)(2n+1) / 6.
